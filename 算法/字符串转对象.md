## 一道简单的手写题

```js
‘a.b.c’ -> {a: {b: {c: null }}}
```

用了个笨办法 面试官也没让优化 直接就到反问环节了

```js
function main(str) {
    let obj = {}
    let data = str.split(".").reverse()
    temp = JSON.parse(JSON.stringify(`{ ${data[0]} : null }`))
    for(let i = 1; i < data.length; i++) {
        obj = JSON.parse(JSON.stringify(`{ ${data[i]} : ${temp} }`))
    }
    return obj
}
console.log(main('a.b.c.d'));
```
